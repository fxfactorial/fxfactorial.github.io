What is React

React is a JavaScript library released by Facebook for creating views. Since being released for web and being a massive success, it has been expanded to other domains, like Mobile (ReactNative)

What is VR -- ReactVR

By buying Occulus, Facebook has been investing heavily in VR and AR technologies. Since React is already the best way to build UIs, it was only logical to extend React to the VR/AR level.

ReactVR is closer to ReactNative than to React on the web as the core components come from ReactNative, like the View, or Text component

ReactVR is built on top the abstractions of three.js which is itself built on top of WebGL which is itself a web abstraction over OpenGL ðŸ˜³

What knowledge is needed? ðŸ’ª

  1. ReactVR should not be your first introduction to React
  2. You should know JavaScript pretty well
  3. Helpful to come from ReactNative background as well
  4. Have a tiny bit of a math background, Linear algebra being the most important

Plays nicely with state management

import React from 'react';
import {AppRegistry, asset, Pano, Text, View, VrButton } from 'react-vr';
import {observable, toJS} from 'mobx';
import {observer} from 'mobx-react';
import global_store from './lib/store.js';

const App =
      observer(class react_vr_housewalk extends React.Component {
  render() {
    const {index, button_positions, current_position} = global_store;
    const renderPositions = button_positions.map((position, idx) => {
      const pos = toJS(position.pos);
      pos[0] -= current_position.pos[0];
      pos[2] -= current_position.pos[2];
      return (
        <VrButton
          key={idx}
          onClick={(e) => {
            console.dir(e);
            const id = e.target - 8;
            const new_position = button_positions[id];
            global_store.current_position = new_position;
          }}
          style={{
            transform : [  {translate : pos}, {rotateX: 90} ],
            borderRadius: 2.5,
            backgroundColor: 'orange',
            height: 5,
            width: 5,
            position: 'absolute'
          }}
        >
        </VrButton>
      )
    });
    return (
      <View>
        <Pano
          source={asset(current_position.photo)}
          onLoad={() => global_store.loaded_pano = true}
        />
    );
